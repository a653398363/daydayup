#### 好记性不如烂笔头

学习的重点之一是习惯的养成，写博客、做总结是学习很好的方法，导致工作两三年后水平停滞不前，最终工资被倒挂，甚至无力跳槽，这其中有一个重要的原因就是在入门的时候没有养成良好的学习习惯，比如看书必须记笔记，有了感悟最好写博客记录一下，发现知识盲区一定要去搜索权威文档而不是听信二手知识等等。

这些习惯我一直都有，并且受益良多。我觉得每个程序员都应该有这些好习惯，因此我用各种办法培养养成这些习惯。
